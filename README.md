# Nombre del Proyecto
Ecommerce-B


# Descripcion
Desarrollo de un sistema basado en microservicios utilizando Spring Boot para la capa backend


## 🛠️ Tecnologías  
- **Backend**: Spring Boot (Java )  
- **Frontend**: Angular 19 
- **Base de datos**: MySQL/PostgreSQL  
- **Herramientas**: Spring Tool Suite
